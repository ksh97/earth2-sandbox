# earth2-sandbox
NVIDIA Earth-2를 이용한 기상예측체계 개발

# 파일 구조
earth2-sandbox/
├─ README.md
├─ .gitignore
├─ .env.example
├─ src/
│  └─ main.py
├─ configs/
│  └─ config.example.yaml
├─ notebooks/
│  └─ README.md
└─ data/
   └─ README.md

# 파일 구조 설명
.env.example: 환경변수 예시만 저장
src/: 실제 Python 코드
configs/: 설정 파일
notebooks/: 실험용 노트북
data/: 데이터 설명만 남기고 실제 데이터는 로컬에만 저장
